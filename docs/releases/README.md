# Notas de release

Esta pasta conterá uma nota para cada versão relevante publicada. A documentação
funcional completa não é copiada para cá; o estado exato de cada versão é
preservado pela tag Git correspondente.

## Modelo

```markdown
# Concentus 1.1.0

Data: AAAA-MM-DD

## Resumo

Descrição curta do valor entregue.

## Novidades

- Funcionalidade e benefício.

## Alterações importantes

- Mudança de comportamento relevante.

## Migração ou ação necessária

- Passos, ou “Nenhuma ação necessária”.

## Limitações conhecidas

- Limitação ainda presente.
```
