# Concentus

Concentus é uma plataforma multi-orquestra, mobile-first, para gestão de membros,
naipes, salas, bibliotecas, repertórios, comunicados e notificações.

O nome de cada orquestra é um dado configurável do tenant, não parte da identidade
do produto. Concentus mantém a mesma identidade visual em toda a plataforma. A
orquestra aparece como contexto por nome, símbolo e imagens permitidas em espaços
controlados, sem alterar cores, tipografia, componentes ou navegação do produto.

> Estado atual: especificação em definição, com versão-alvo 1.0.0. Ainda não há
> implementação.
